This comprehensive solution showcases the synergy of diverse technologies to address agricultural challenges, offering accurate yield predictions and profitable crop recommendations.

College Enquiry Chatbot

The College Enquiry Chatbot project, executed from Aug 2021 to Nov 2021, employed a robust tech stack comprising Flask, NodeJS, Dialogflow, SQLite, HTML5, CSS, JavaScript, Android Studio, and Natural Language Processing (NLP). This cross-platform college chatbot, powered by Dialogflow NLP, significantly enhanced user experience by providing real-time and precise information. The backend, deployed using DialogFlowJS and NodeJS with API service key, demonstrated scalability by supporting over 1000 concurrent users. A meticulous test plan was formulated, evaluating accuracy, performance, and scalability, resulting in an impressive 10-second response time. This project exemplifies the seamless integration of various technologies to create an efficient and scalable college enquiry chatbot solution.
